Welcome to MptcpAnalyzer's documentation!
=========================================


This is the manual of mptcpanalyzer, a python based linux tool to help plot some characteristics of an multipath TCP connection based on network traces (\*.pcap files).

This document is written in `reStructuredText <http://docutils.sourceforge.net/rst.html>`_ for `Sphinx <http://sphinx.pocoo.org/>`_ and is maintained in the
``doc/`` directory of the package source code.
